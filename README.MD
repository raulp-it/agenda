Agenda-Dbase-III-a-Python
Agenda personal desarrollada originalmente en 1995 utilizando DBase III Plus sobre MS-DOS 5.0.

Este proyecto nació como una herramienta para administrar contactos, teléfonos, direcciones y anotaciones personales en una época donde las agendas electrónicas y los teléfonos inteligentes aún no existían. Fue uno de mis primeros programas completos y representa una etapa importante de mi aprendizaje como programador.

El desarrollo comenzó sobre una PC XT equipada con un procesador NEC V20, 1 MB de RAM, disco rígido Seagate ST-238R de 60 MB, unidad de disquete de 5,25" y monitor monocromático Hércules. El lenguaje elegido fue DBase III Plus, que fue mi puerta de entrada al mundo de las bases de datos y la programación.

Aprendí DBase III a partir de un libro prestado de biblioteca que conseguí gracias a mi tío Egidio. Como el libro debía devolverse y no podía fotocopiarse correctamente debido al tipo de impresión utilizado, fue necesario avanzar rápidamente y aprovechar al máximo el tiempo disponible para estudiar y programar.

Los archivos incluidos en este repositorio corresponden al código fuente original conservado desde aquella época. Se publican con fines educativos, históricos y de preservación informática, mostrando cómo se desarrollaban aplicaciones de gestión personal en computadoras domésticas de los años noventa.

Más que una simple agenda, este proyecto es una pequeña cápsula del tiempo que refleja una etapa de aprendizaje, experimentación y pasión por la informática.
